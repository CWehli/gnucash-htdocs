récupération du code source
Récupération des plus récentes sources via CVS:

Une version en lecture seule de l'arborescence de cvs est disponible sur le net. Pour y accéder, premièrement, logger vous(login), comme ça: 

        cvs -d :pserver:cvs@cvs.gnucash.org:/home/cvs/cvsroot login

Le mot de passe (password) est "guest". Pour récupérer une copie du source, faites un

        cvs -d :pserver:cvs@cvs.gnucash.org:/home/cvs/cvsroot checkout gnucash 

Notez que  plusieurs versions peuvent être accédé avec des paramètres (tags). Par exemple, le paramètre  xacc-10b17 vous récupèrera la version 1.0.17 et le paramètre xacc-11b6 vous récupèrera la version 1.1.6 . En particulier, le dernier code source dans les séries  1.0.x (les séries stable) est disponible sur la branche xacc-10-patch. Pour les enregistrements des historiques, vous pouvez voir le source initial de Robin Clark d'Octobre 1997 à xacc-09a. Les choses ont changés *beaucoup* depuis lors. 

Récupérer les plus récentes sources via FTP:

Si vous ne voulez pas vous ennuyez  avec CVS , vous pouvez juste télécharger la dernière capture de l'arborescence de CVS à partir du site FTP . ftp://ftp.gnucash.org/pub/
