GnuCash

GnuCash ist ein Programm zur Verwaltung persönlicher Finanzen. Eine scheckbuchähnliche Oberfläche erlaubt die Eingabe und das Verfolgen von Bankkonten, Aktien und sogar Währungshandel. Die Oberfläche ist einfach und leicht zu bedienen, verwendet aber das Prinzip der doppelten Buchführung.

Funktionen von GnuCash sind:

  • Ein einfaches Benutzerinterface: Wer einen Kontoauszug versteht, kann GnuCash verwenden.
  • QuickFill: wenn ein Beschreibungstext für eine Buchung mit einem vorangehenden Text identisch ist, kann er mit <TAB> automatisch komplettiert werden (z.B. für regelmäßige Buchungen wie das Gehalt).
  • Handhabung von Aktien- und Fondsportfolios: der aktuelle Stand kann einzeln (eine Aktie = ein Konto) oder als Portfolio (alle Aktien = ein Konto) verwaltet werden.
  • Verschiedene Währungen und Devisenhandel: Mehrere Währungen werden unterstützt und können ge- und verkauft werden. Währungsbewegungen auf den Konten werden ausgeglichen.(Einige Teile von Devisenhandel sind aber noch nicht vollständig implementiert.)
  • Import von Dateien im Quicken Interchange Format (QIF).
  • Berichte: Anzeigen von Gewinn & Verlust oder Portfolios, auch in HTML
  • Hierarchie von Konten: Ein Haupt-Konto kann eine Hierarchie von Unter-Konten haben. Das macht es möglich, ähnliche Konten (Bargeld, Spar-, Aktien-Konten) in einem Hauptkonto zusammenzufassen.
  • Split-Buchungen: Eine einzelne Buchung kann in mehrere Teile zerlegt werden, wie z.B. Steuern, Gebühren, usw.
  • Erlaubt das Aktualisieren von Aktien- und Fonds-Kursen über das Internet, auch automatisch.
  • Geschrieben in "C" mit eingebetteter Unterstützung von Scheme durch "guile".
  • GnuCash steht unter der GPL (GNU General Public License)

Download

Quelltext- und ausführbare Programmpakete sind unter folgenden URLs erhältlich. Die derzeitigen stabilen Versionen haben Versionsnummern 1.4.x, die Entwicklungsversionen haben Versionsnummern 1.5.x.

  • ftp://ftp.gnucash.org/pub/gnucash/
  • http://download.sourceforge.net/gnucash/

Die jeweils aktuellste Entwicklungsversion kann vom GnuCash CVS Server bezogen werden. Dazu muß man sich zuerst am CVS Server anmelden:

  • cvs -d :pserver:cvs@cvs.gnucash.org:/home/cvs/cvsroot login

Das Passwort ist 'guest'. Den Quellcode bekommt man dann mit:

  • cvs -d :pserver:cvs@cvs.gnucash.org:/home/cvs/cvsroot checkout gnucash

Eine einmal ausgecheckte Version von GnuCash bringt man folgendermaßen auf den neuesten Stand:

  • cd <GnuCash-Verzeichnis>
  • cvs up -Pd
